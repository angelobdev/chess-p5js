# Chess P5JS

A simple remake of a great game!

---

## How to run

1. Install dependencies:

   ```bash
   bun install
   ```

2. Run:

   ```bash
   bun start
   ```

Note:  
Bun is not strictly necessary, but it has been used to build this project... You can use NPM as well!
